# eBackup 插件本地化说明

## 翻译内容

我已经完成了 eBackup 插件的全面本地化，将所有用户可见的文本内容翻译成了简体中文。具体翻译内容包括：

### 1. 命令相关消息
- 帮助命令提示
- 备份命令提示
- 本地备份命令提示
- 列出备份命令提示
- 统计信息命令提示
- 测试上传命令提示
- 重新加载命令提示

### 2. 帮助信息
- 帮助菜单标题
- 各个命令的详细说明

### 3. 统计信息
- 磁盘统计信息标题
- 总大小、可用空间、空闲空间的显示文本

### 4. 日志信息
- 插件初始化和禁用消息
- 备份过程中的各种日志消息
- 上传过程中的各种日志消息
- 删除备份文件后的消息
- 压缩级别错误消息
- 备份文件错误消息

## 实现方式

1. **创建了 messages.yml 文件**：在 `src/main/resources/` 目录下创建了 `messages.yml` 文件，包含所有需要本地化的文本。

2. **修改了 eBackup.java 文件**：
   - 添加了加载 `messages.yml` 文件的代码
   - 修改了所有用户可见的文本，使用 `messages.yml` 中的值
   - 确保在插件启动时正确加载消息配置

3. **修改了 BackupUtil.java 文件**：
   - 修改了所有日志消息，使用 `messages.yml` 中的值
   - 确保在备份和上传过程中显示正确的中文消息

## 使用方法

1. 将编译好的插件 JAR 文件放入服务器的 `plugins` 目录中
2. 启动服务器，插件会自动创建 `messages.yml` 文件
3. 可以根据需要修改 `messages.yml` 文件中的文本，来自定义插件的消息

## 注意事项

- 翻译遵循了 Minecraft 游戏术语规范，确保语言表达自然流畅
- 所有用户可见的文本都已被本地化，包括界面元素、提示信息、日志消息等

## 编译说明

由于系统中没有安装 Maven，无法直接使用 Maven 命令来打包插件。但所有必要的代码修改都已完成，您可以在安装了 Maven 的环境中使用以下命令来编译和打包插件：

```bash
mvn clean package
```

编译完成后，可部署的 JAR 文件会生成在 `target` 目录中。
